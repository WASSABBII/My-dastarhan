## 2. WHATSAPP API — РЕАЛЬНЫЕ ЦЕНЫ (после июля 2025)

### Новая модель (с 1 июля 2025):
- **Service-сообщения** (ответ клиенту в 24ч окне) — **БЕСПЛАТНО и безлимитно**
- **Utility** (напоминания вне 24ч окна) — платно по тарифу
- **Marketing** — платно всегда

### Цены по регионам:
| Регион | Utility/сообщение | Marketing/сообщение |
|--------|------------------|---------------------|
| Казахстан ("Other") | $0.0077 | $0.0604 |
| Россия | $0.04 | $0.0802 |

### Meta напрямую vs Twilio:
| Параметр | Meta Cloud API | Twilio |
|----------|---------------|--------|
| Utility (КЗ) | $0.0077/msg | $0.0127/msg |
| Utility (РФ) | $0.04/msg | $0.045/msg |
| Сложность интеграции | Средняя | Низкая |
| Sandbox для тестов | Есть | Удобнее |
| **Рекомендация для MVP** | — | **ДА** |

**Важный сценарий:** Если клиент пишет при бронировании — открывается 24ч окно. Все ответы и напоминания внутри этого окна = **БЕСПЛАТНО**. Это очень выгодно.

## 3. СТЕК ТЕХНОЛОГИЙ

| Слой | Технология | Почему |
|------|-----------|--------|
| Frontend | Next.js 14+ | SSR, i18n встроено, Vercel деплой |
| Backend | NestJS (TypeScript) | Один язык с фронтом, WhatsApp SDK на Node.js |
| База данных | PostgreSQL (Supabase) | Auth + Realtime из коробки |
| Очереди | BullMQ + Redis | Delayed jobs для напоминаний за 1ч |
| Хостинг Frontend | Vercel | Hobby бесплатно |
| Хостинг Backend | Railway | $5–20/мес |
| WhatsApp | Twilio | Проще SDK, хорошая документация |

**Почему NestJS, а не FastAPI:**
- Официальный WhatsApp SDK — на Node.js
- BullMQ нативно работает с Node.js
- Один язык (TypeScript) на фронте и бэке

## 4. FEASIBILITY — ВСЁ РЕАЛИЗУЕМО

| Функция | Сложность | Примечание |
|---------|-----------|------------|
| Бронирование через сайт | Низкая | Стандартный CRUD |
| Авто-напоминание за 1ч через WhatsApp | Средняя | BullMQ delayed job + Twilio |
| Ответ Да/Нет → смена статуса | Средняя | WhatsApp webhook, парсинг входящего |
| Kanban (drag & drop) | Средняя | @dnd-kit + WebSocket real-time |
| Дашборд статистики | Низкая | Recharts, SQL-агрегации |
| Мультиязычность RU/KZ/EN | Низкая | Next.js i18n встроено |
| Флаг предоплаты + предупреждение | Низкая | Boolean поле в БД |

**Нет ни одного технического блокера.**
Сложнее всего — WhatsApp webhook + edge cases (клиент пишет что-то кроме Да/Нет).

## 6. НАШИ ПЛЮСЫ И МИНУСЫ

### ПЛЮСЫ:
1. **Казахстан** — незанятая ниша, ни один конкурент там не работает
2. **WhatsApp-first** — у OpenTable/Resy нет, у ReMarked слабое
3. **Цена** — можем предложить $30–80/мес vs $199–499 у глобальных
4. **Локализация** — RU + KZ + KZT; глобальные этим не занимаются
5. **Простой онбординг** — крупные требуют недели; MVP за 1 день
6. **No-show проблема** — авто-напоминания снижают no-show на 30%

### МИНУСЫ:
1. **Нет сети дайнеров** — у OpenTable 150 млн в базе; у нас 0
2. **Доверие** — нужны первые кейсы и отзывы
3. **B2B продажи** — цикл 2–6 недель, нужны люди
4. **Нет POS-интеграций** — iiko, R-Keeper сложно для малой команды
5. **Верификация Meta** — занимает 1–4 недели до запуска WhatsApp API
