# Dastarkhan

## Запуск

### Бэкенд

```bash
cd apps/backend
npm install
npm run start:dev
```

Сервер запустится на `http://localhost:3005`

### Фронтенд

```bash
cd apps/frontend
npm install
npm run dev
```

Сайт откроется на `http://localhost:3000`

### Seed (тестовые рестораны)

```bash
cd apps/backend
npm run seed
```
